Bonjour à tous, 

Bienvenue sur notre pokedex Python X Django !

L'utilisation est simple : 
1. Clonez le git avec la commande : git clone https://github.com/Seikanuma/PokeDjango.git
2. Déplacez vous dans le dossier du projet : cd PokeDjango/PokeDjango
3. Lancez le projet à l'aide de la commande python manage.py runserver
4. Une fois l'application lancée, vous arrivez sur la liste des 151 premiers Pokémons.
5. A partir de cette liste, vous pouvez cliquer sur un Pokémon pour voir le détail, et changer de Pokémon en descendant sur la page.
6. En restant sur l'accueil il est aussi possible de changer de page avec les flèches en bas de page (il y a 6 pages au total).

Amusez vous bien ! :D
